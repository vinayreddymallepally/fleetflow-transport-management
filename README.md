# FleetFlow Transport Management System

FleetFlow is a full-stack MERN transport management system designed to manage vehicles, shipments, logistics workflow, and analytics dashboards.

## Features

- JWT Authentication
- Vehicle Management
- Shipment Management
- Workflow Tracking
- Dashboard Analytics
- MongoDB Atlas Integration
- Responsive UI
- Render + Vercel Deployment

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios
- React Router

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication
